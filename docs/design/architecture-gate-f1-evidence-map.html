Any drift fails with an instruction to update both the JSON baseline and this evidence map in the slice that intentionally moved the surface.

Rules

Counting contract

Mechanical counts

  • Public declarations are nonblank non-include snapshot lines; overloads and constructors count.
  • Binding methods are named .def("...") registrations on py::class_<PyCircuit>; top-level module.def exports are separate.
  • Caller strata count direct lexical .name( and ->name( tokens for root snapshot names.
  • Hidden footprint injections count only production src/python calls to volt::builtin_footprint_library().
  • Goldens freeze normalized JSON shape plus raw byte length and SHA-256.

Intentional exclusions

  • Generated docs and architecture HTML are not product call-site evidence.
  • Indirect aliases, templates and function pointers are outside the lexical caller count.
  • Tests and examples are excluded from hidden resolver injection counts but included where caller strata explicitly name tests.
  • This gate does not enforce D1-D3 designs or forbid approved future movement.
