Volt is a modern electronics design kernel for tools and scripts that need circuit data they can trust.

Most EDA workflows start with drawings. Volt starts with structured intent: component definitions, component instances, pins, nets, selected physical parts, schematic projection data, PCB placement/routing data, validation diagnostics, and deterministic JSON artifacts.

## Core idea

```text
Invalid kernel state should be impossible.
Bad circuit design should be diagnosable.
```

Python is an authoring surface over kernel-owned state. It makes circuit generation pleasant, but it does not become a second circuit database.

## What Volt writes

Volt currently produces deterministic logical circuit JSON, schematic projection JSON, schematic SVG, PCB projection JSON, PCB SVG, and validation reports. These artifacts are intended to be easy to diff, test, and feed into future UI or adapter layers.
