## Design root

| Export | Purpose |
| --- | --- |
| `volt.Design` | Root handle for one kernel-owned logical circuit plus schematic and PCB projections. |

Common methods: `net`, `nets`, `R`, `C`, `CP`, `L`, `LED`, `diode`, `switch`, `crystal`, `test_point`, `connector_1x01`, `connector_1x02`, `connector_1x03`, `regulator`, `op_amp`, `define_component`, `define_module`, `instantiate`, `schematic`, `board`, `validate`, `validate_for_pcb`, `to_json`, and `write`.
